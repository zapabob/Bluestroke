# Sound Files

This folder contains keyboard sound presets for Bluestroke.

## Folder Structure

- `Blue/` - Blue switch mechanical keyboard sounds (clicky)
- `Brown/` - Brown switch mechanical keyboard sounds (tactile)
- `Red/` - Red switch mechanical keyboard sounds (linear)
- `MacBook/` - MacBook style keyboard sounds
- `Custom/` - User-defined custom sounds

## Adding Custom Sounds

1. Place `.wav` audio files in the `Custom` folder
2. In Bluestroke settings, select "Custom" as your sound preset
3. Browse to the Custom folder location

## Sound File Requirements

- Format: WAV (PCM)
- Sample Rate: 44100 Hz (recommended)
- Channels: Stereo (2 channels) preferred
- Bit Depth: 16-bit or 32-bit float
- Duration: Short sounds (< 500ms recommended)

## Obtaining Sound Files

You can obtain keyboard sound files from:
- Record your own mechanical keyboard
- Use royalty-free sound libraries
- Create synthesized sounds using audio software

Note: The included sound files are placeholders. Replace them with actual keyboard sounds for the full experience.
